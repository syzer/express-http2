# WAT

node.js + express + http2

# HOW

To watch awesome doge over push http2 do:

```bash
npm i
npm start
```
[open https link](https://127.0.0.1:4002/)

### Please note
normal express handlers like

```js
res.status(200).json({})
```

or

```js
res.sendFile(__dirname + '/doge_wat.png')
```

also work
